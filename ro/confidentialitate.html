Confidenţialitate

Date colectate

Bitcoin.org colectează loguri server anonimizate. Aceste loguri includ adrese IP cu ultimul byte înlocuit, data accesării, pagina solicitată, user agent şi referer url. Bitcoin.org nu colectează date folosing cookie-uri.

Folosirea de date

Datele colectate sunt folosite pentru a oferi statistici publice transparente.
