Privacidade

Dados coletados

Bitcoin.org coleta registros anônimos do servidor. Esses registros incluem endereços IP com último byte substituído, o tempo da visita, página solicitada, agente de usuário e url referente. Bitcoin.org não coleta dados utilizando cookies.

Uso de dados

Os dados coletados são usados para fornecer estatísticas públicas transparentes.
